## 2 后台功能

### 2.1 后台登录页面功能制作

>  将所有的接口地址统一保存在config.js文件中，方便后期维护。

- 结构的修改：
    - 登录按钮是submit功能。
  - 希望使用ajax，应改为普通按钮。或者使用事件对象阻止元素的默认行为
  - 如a的跳转和submit的跳转
  - e.preventDefault()

- 功能简介：
  1. 点击登录按钮后，获取用户名和密码
  2. 检测获取的内容是否为空
  3. 如果不为空，通过接口发送给服务器端检测
  4. 根据检测结果进行对应处理
     - 成功： 跳转到首页
     - 失败： 进行相关提示
- 实现功能
  1. 给登录按钮添加点击事件
  2. 获取用户名和密码并进行检测
  3. 如果不为空则发送ajax请求
  4. 改进接口地址，创建一个tool文件夹放配置文件,例如将接口地址统一保存在config.js中。
